# 程序员人格测试

> 通过简单的选择题，测试你在程序员职场生态中的真实人格定位

## 🌐 在线体验

**👉 https://lukelmouse-github.github.io/programmer-persona-test/**

## 简介

本测试灵感来源于：
- 贴吧老哥的真实吐槽
- 程序员职场生态观察
- 阿里巴巴职级体系参考

## 特点

- ✅ **快速测试**：只需 8 道题，1 分钟完成
- ✅ **职场为主**：70% 职场维度 + 30% 技术维度
- ✅ **接地气**：融入贴吧、内涵段子风格，生动有趣
- ✅ **长截图分享**：一键生成精美分享图片
- ✅ **UI 美观**：深色主题 + 霓虹渐变色
- ✅ **纯前端**：无需后端，打开即用

## 使用方法

1. 克隆或下载本项目
2. 用浏览器打开 `index.html`
3. 开始测试

## 部署

可以直接部署到 GitHub Pages：

```bash
# 在你的 GitHub 仓库中
git add .
git commit -m "Add programmer persona test"
git push origin main
```

然后在仓库设置中开启 GitHub Pages 即可。
